Projects

  • WorkIt
    WorkIt allows users to securely keep track of their workouts with an interactive calendar. Add workouts by clicking on a calendar day, and edit them with a click or by dragging and dropping them to a new day. The front end is built with Angular.js (specifically, the Angular-UI suite), and the back end is built in Rails 5 and relies on a PostgreSQL database. User authentication is implemented with bcrypt and json web tokens.
  • Endeavor
    Endeavor gives users a platform to discover the Digital Public Library of America's breadth of images. Using the DPLA's API, the app displays images chosen at random from the library, based on randomly generated search terms. Users can save images to their favorites list, and can click on a particular image to find more images like it. The front end uses AJAX to query the API, and Angular.js to display the data. The back end is built with Node.js and Express.js, and relies on a MongoDB database.
  • Cheerups
    Cheerups is a social media platform on which users can create, read, update, and delete posts ('cheerups') meant to brighten your day. Users can also view all cheerups posted by a specific user, view all cheerups by popularity, upvote cheerups, and get inspired with a random cheerup from the collection. The front end is rendered in EJS, while the back end is built with Node.js and Express.js. The app relies on a MongoDB database.
